# banking-management-system-GUI-java
A Java-based Bank Management System with Swing GUI, implementing OOP concepts, file handling, and core banking operations such as account creation, deposits, withdrawals, and fund transfers
 Features:
- Create account
- User login
- Admin login
- Deposit money
- Withdraw money
- Transfer funds
- Transaction history
- Search client by CNIC
- Remove client
- Calculate total bank balance
- File handling using text files

## Technologies Used:
- Java
- Java Swing
- OOP 
- File Handling
- Error Handling
